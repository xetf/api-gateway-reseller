## 等级能力

套餐模板、租户/代理商、月结/发票能力已下线，数据库表和后台接口同步移除。

现有口径：

- 用户管理直接设置每个用户的等级、模型白名单、账号限流和并发限制。
- 调度与分发页面维护等级预设。
- IP 等级规则在调度与分发页面维护，命中时优先级高于用户等级。

## 运维告警

新增 `operational-alerts` 服务：

- 采集 Redis、数据库、PM2、模型池、上游 Key、API Key 并发状态。
- 生成 info / warning / critical 告警。
- 支持外部 webhook 推送。
- 支持后台配置启停、最低级别、推送间隔、附加提醒文本。
- 支持测试推送。

## 剩余风险和人工验证边界

自动验证已覆盖类型检查、构建、迁移、服务健康和后台页面响应。以下流程需要管理员登录态、真实上游 Key 或业务测试账号才能完整验收：

- 创建套餐模板并套用到真实用户。
- 创建租户/代理商并分配用户。
- 设置月结信用额度并创建发票。
- 配置外部 webhook 并确认通知平台实际收到消息。
- 使用真实 API Key 发起非流式和流式模型调用。
- 验证真实上游失败时的重试、公告返回和渠道惩罚。
